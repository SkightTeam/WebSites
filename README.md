# Skight I-Tech Inc.

Professional portal site for Isaac Wang / Skight I-Tech Inc.

The site presents modern software development, workflow automation, Microsoft 365 management, AI knowledge workflows, and recent project work.

## Development

This repository is a lightweight static site. Source HTML lives under `source/`, shared styling lives in `source/assets/portal.css`, and the build script copies the production site into `dist/`.

```bash
npm install
npm run build
npm run serve
```

The generated static site is written to `dist/`.

## Deploy

```bash
npm run deploy
```
